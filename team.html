Team Management

For effective team management, we used a mix of tools, listed below:

Microsoft Teams. This served as the primary communication channel for our team. It was used to hold weekly meetings and provide regular updates to all team members.
GitHub. We created Github repositories (iOS application, web server, Webots) in order to coordinate all the code written for the development of the system.
Overleaf. We used Overleaf to coordinate and work on our Group Plan and the Demo reports.
